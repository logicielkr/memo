# [Graha 응용프로그램]메모장

## 1. about

Graha를 활용한 메모장 프로그램이다.  

## 2. 기능

이 프로그램은 1) MarkDown 및 Highlight(Line Numbering 포함) 기능, 2) 자동저장 기능 3) 이력관리 기능을 포함한다.

## 3. 실행환경

이 프로그램은 다음과 같은 환경에서 개발되고 테스트 되었다.

- 서버 환경
	- Apache Tomcat 7.x 이상
	- PostgreSQL 9.1.4 이상
	- JDBC 4.0 이상
	- Graha 0.5.0.6 이상
